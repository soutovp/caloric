## **Como será o APP ?**
Com o objetivo de calcular o gasto calórico a partir de informações pré definidas no app
## **Quais os calculos ?**
- Base IMC para verificar se está acima do peso ou não.
## **Qual será os dados a serem calculados ?**
- Peso;
- Altura;

## **Quais as opções poderemos aplicar para atingir os seguintes objetivos :**

- Ganho de Massa;
- Perda de Peso;
- Manter Peso;
- Perda de Gordura Focando Ganhar Massa;
